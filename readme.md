# Channels to eat to.

I'm currently building a model trained on a dataset (that I created!) of video-essay style channels that people like to watch when eating a meal alone! This app will "recommend" similar channels based on an input channel and several other parameters that users may want to specify. I hesitate to use the word "recommend" as this is not a "true" recommendation system (there is no utility matrix as there is no user data, rather I'm just finding the nearest neighbours). 

The repository is messy right now, but I'll get around to cleaning it up very soon!
